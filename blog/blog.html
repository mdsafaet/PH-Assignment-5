What is the Document Obect Model (DOM)

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects; that way, programming languages can interact with the page.
A web page is a document that can be either displayed in the browser window or as the HTML source. In both cases, it is the same document but the Document Object Model (DOM) representation allows it to be manipulated. As an object-oriented representation of the web page, it can be modified with a scripting language such as JavaScript.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated. It allows us to attach a single event listener for elements that exist now or in the future.

How do you manipulate an element's attributes and styles using the DOM

To manipulate an element's attributes and styles in the DOM, you can access and modify them directly. For attributes, retrieve or set values dynamically to update elements like images or inputs. Styles can be adjusted by changing CSS properties, allowing you to create responsive, interactive visual changes on the page.
